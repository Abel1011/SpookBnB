# SpookBnB - Product Overview

SpookBnB is a horror-themed vacation rental landing page with a dual-mode experience:

- **Light Mode**: A welcoming, luxury cabin rental website with warm aesthetics
- **Dark Mode**: A terrifying horror experience with jump scares, glitch effects, and an escape room puzzle

## Core Concept
The site appears as a normal vacation rental but transforms into a haunted experience when users toggle to dark mode. Users can solve riddles hidden throughout the dark mode to "escape."

## Key Features
- Theme toggle that triggers dramatic visual/audio transitions
- Flashlight overlay effect in dark mode (cursor reveals content)
- Hidden clues and escape room mechanics
- Ambient horror audio and sound effects
- Animated ghost text, cursor trails, and jump scares
- Reservation system with form persistence
- Gallery modal for property images
- Review cards that transform between wholesome and horrifying content
